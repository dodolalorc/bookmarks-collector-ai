# 星花鸟（Nuci）

星花鸟（Nuci）是一个基于 Plasmo 构建的浏览器扩展 AI 助手，专注于网页知识抓取、页面内容整理和书签归档。

它不是“自动替你做决定”的黑盒，而是把网页里的正文、片段、标签和整理建议提炼出来，再由你确认是否保存、如何保存、保存到哪里。

后续我会补充项目截图到这里。

## 适合做什么

- 浏览文章时，快速抓取重点段落，沉淀成可管理的知识片段。
- 把一篇网页内容整理成适合继续投喂大模型的结构化材料。
- 用 AI 或本地规则辅助推荐书签归档位置。
- 管理历史抓取内容、知识记录和配置备份。

## 主要功能

### 1. 页面知识抓取面板

在网页右侧提供一个内嵌式抓取面板，打开后会像浏览器开发者工具一样压缩页面可视宽度，而不是简单遮挡网页内容。

支持：

- 选中文本后直接加入当前抓取面板。
- 开启框选模式，点击页面任意内容区域抓取文本。
- 对已抓取片段做逐条分析或批量分析。
- 拖动边缘调整面板宽度。

### 2. AI 页面整理弹窗

页面右侧下方悬浮球会打开独立的“AI 页面整理”弹窗，用来集中处理当前网页正文，而不是和抓取面板耦合在同一个抽屉里。

支持：

- 编辑标题、作者、日期、网址等元信息。
- 编辑 AI 提示语，指定整理视角。
- 选择当前使用的模型。
- 一键整理网页正文，输出更适合 AI 使用的内容。

### 3. 全文模式 / 分段模式

AI 页面整理支持两种内容模式：

- 全文模式：把抓取到的正文作为一个整体输入框处理，适合快速人工修订后直接整理。
- 分段模式：按网页自然段落拆成多个独立片段，每一段都可以单独勾选是否参与整理。

这个模式尤其适合处理正文中掺杂广告、推荐阅读、版权尾注、导流文案的页面。

### 4. 智能选取

在分段模式下，可以使用“智能选取”能力，让大模型自动筛出更可能属于正文主线的段落，并尽量剔除无关内容，例如：

- 页面中的广告插段
- 无关推荐阅读
- 导流按钮文案
- 与正文关系较弱的尾注或说明

如果当前没有配置模型，也会有明确提示，引导你前往模型配置页面完成设置。

### 5. 书签推荐与整理

星花鸟（Nuci）会结合当前网页信息、备注、标签和已有书签结构，生成推荐的收藏目标。

支持：

- 本地规则推荐
- 接入 OpenAI 兼容接口后的 AI 增强推荐
- 历史书签批量整理
- 用户确认后再真正写入浏览器书签

### 6. 知识记录沉淀

当你确认整理或收藏后，扩展可以把页面标题、网址、目录路径、标签、备注和选中文本沉淀到本地知识记录中，方便后续回看和检索。

### 7. 配置导入 / 导出

现在支持导出和导入备份。

导出内容包括：

- 模型配置
- 已沉淀的知识记录
- 页面抓取草稿
- 抓取内容收藏

导入时会恢复扩展内部数据，但不会直接改写你浏览器里现有的书签树。

### 8. 快速开始页

管理页新增“快速开始”标签页，用来介绍各个部件的作用和推荐使用流程，方便第一次接触扩展时快速上手。

## 使用方式

### 基础流程

1. 打开任意网页。
2. 点击页面右侧上方悬浮球，打开“页面知识抓取”。
3. 划线选择内容，或开启框选模式抓取页面片段。
4. 点击下方悬浮球，打开“AI 页面整理”。
5. 选择全文模式或分段模式，按需编辑正文与提示语。
6. 如有需要，使用“智能选取”过滤无关段落。
7. 执行一键整理，得到适合继续输入 AI 的页面材料。
8. 在弹出页或管理页中继续完成书签归档和知识管理。

### 首次使用建议

1. 先进入“模型配置”填写 `Base URL`、`API Key` 和模型名。
2. 再打开任意文章页测试抓取和整理功能。
3. 最后导出一份备份，方便之后迁移或恢复配置。

## 界面组成

### Popup

用于快速查看当前页面、补充标签和备注、生成收藏建议、执行书签归档。

### 页面内悬浮入口

网页中会有两个悬浮球：

- 上方：打开页面知识抓取面板
- 下方：打开 AI 页面整理弹窗

两个入口互相独立，不会再出现联动展开或关闭互相干扰的问题。

### 管理页

包含以下几个标签页：

- 快速开始
- 模型配置
- 历史整理
- 知识笔记

## 模型配置说明

当前模型配置完整后，以下能力会使用大模型：

- AI 页面整理
- 智能选取
- AI 增强书签推荐
- 抓取片段分析

如果没有配置模型，扩展仍可继续使用本地抓取、本地推荐和内容管理能力，并在需要模型时给出明确提示。

## 技术说明

- 扩展框架：Plasmo
- 前端界面：Vue 3
- 运行方式：浏览器扩展本地运行
- 数据存储：`chrome.storage`

## 本地开发

```bash
pnpm dev
```

开发时将生成的扩展目录加载到浏览器即可。

## 构建

```bash
pnpm build
```
