# 星花鸟（Nuci）

星花鸟（Nuci）是一个基于 Plasmo 构建的浏览器扩展，核心目标不是“自动替你整理一切”，而是把网页阅读过程里的重点内容抓出来、整理好、再稳妥地归档到书签和本地知识记录中。

当前版本已经收敛为一条明确主线：

1. 在网页中抓取重要片段
2. 用 AI 或本地规则整理当前页面
3. 推荐书签归档位置并由用户确认写入
4. 把整理结果沉淀到本地知识记录中，便于后续回看

## 适合做什么

- 浏览文章时，快速摘录关键段落，避免“看过就忘”
- 把一篇网页整理成更适合继续投喂大模型的内容
- 在收藏网页前先理解内容，再决定归档到哪里
- 持续积累本地知识记录，按标签、目录和关键词回看

## 当前功能

### 1. 页面知识抓取

网页右侧提供内嵌式抓取面板，打开后会像浏览器侧边工具一样为页面预留宽度，而不是直接遮挡正文。

支持：

- 选中文本后直接加入当前页面草稿
- 开启框选模式，点击页面元素抓取文本
- 对抓取片段逐条分析或批量分析
- 拖动侧边面板边缘调整宽度

抓取到的片段会进入当前页面草稿，并同步沉淀到“内容整理”工作区，方便后续统一处理。

### 2. AI 页面整理

页面右侧下方悬浮入口会打开独立的“AI 页面整理”弹窗，用来集中处理当前网页正文。

支持：

- 编辑标题、作者、日期、网址等元信息
- 编辑 AI 提示语，指定整理视角
- 选择当前模型
- 一键整理当前页面内容

### 3. 全文模式 / 分段模式

AI 页面整理支持两种处理方式：

- 全文模式：将抓取到的正文作为整体输入，适合快速修改后直接整理
- 分段模式：按段落拆分正文，可逐段勾选是否参与整理

分段模式适合处理正文里混入广告、导流文案、相关推荐、版权说明等噪音内容的页面。

### 4. 智能选取

在分段模式下，可以使用“智能选取”能力，让模型自动筛出更接近正文主线的段落，尽量排除页面噪音。

如果当前没有完成模型配置，扩展会明确提示，并引导你进入模型配置页。

### 5. 当前页面书签归档

Popup 以“当前页面归档”为核心。

支持：

- 读取当前页面标题、摘要、选中文本等上下文
- 手动补充标签和备注
- 基于现有书签结构生成归档推荐
- 在用户确认后写入浏览器书签

如果已接入模型，可获得 AI 增强推荐；如果未配置模型，也可以继续使用本地规则推荐。

### 6. 内容整理工作台

管理台中的“内容整理”页用于集中管理抓取结果和高阶整理任务。

支持：

- 管理抓取片段收藏夹
- 编辑、移动、删除抓取内容
- 查看片段原文和分析结果
- 对已有历史书签生成批量迁移建议

其中历史书签整理属于高级功能，适合在确认推荐结果后再批量操作。

### 7. 知识笔记库

当你确认归档后，扩展会把页面标题、网址、目录路径、标签、备注和选中片段沉淀到本地知识记录中。

支持：

- 查看已沉淀的知识记录
- 按标题、网址、目录、标签、备注搜索
- 查看目录分布和标签频率

### 8. 模型配置与备份

管理台中的“模型配置”页负责管理模型和导入导出。

支持：

- 配置多个 OpenAI 兼容模型
- 选择当前活动模型
- 自定义 Prompt 模板
- 导出本地备份
- 导入本地备份

导出内容包括：

- 模型配置
- 已沉淀的知识记录
- 页面抓取草稿
- 内容整理收藏夹

导入时会恢复扩展内部数据，但不会直接改写浏览器已有书签树。

## 使用方式

### 推荐流程

1. 进入管理台，先完成模型配置
2. 打开任意网页
3. 点击右侧上方悬浮球，抓取重要片段
4. 点击右侧下方悬浮球，整理当前页面正文
5. 在 Popup 中补充标签和备注，生成归档推荐
6. 确认后写入浏览器书签
7. 后续在“知识笔记”中回看整理结果

### 首次使用建议

1. 先填写 `Base URL`、`API Key` 和模型名
2. 选一篇结构较清晰的文章，测试抓取和 AI 整理
3. 完成一次书签归档，确认知识记录是否正常生成
4. 导出一份备份，方便之后迁移或恢复

## 界面组成

### Popup

用于处理“当前页面归档”：

- 查看当前页面上下文
- 补充标签和备注
- 生成目录推荐
- 确认写入书签

### 页面内悬浮入口

网页中会有两个悬浮球：

- 上方：打开页面知识抓取
- 下方：打开 AI 页面整理

两个入口独立工作，分别服务于“抓取内容”和“整理内容”两件事。

### 管理台

当前管理台保留三个入口：

- 模型配置
- 内容整理
- 知识笔记

## 模型配置说明

模型配置完整后，以下能力会使用大模型：

- AI 页面整理
- 智能选取
- AI 增强书签推荐
- 抓取片段分析

如果没有配置模型，扩展仍然可以继续使用：

- 本地抓取
- 本地推荐
- 内容整理
- 知识记录管理

## 技术说明

- 扩展框架：Plasmo
- 前端界面：Vue 3
- 运行方式：浏览器扩展本地运行
- 数据存储：`chrome.storage`

## 本地开发

```bash
pnpm dev
```

开发时将生成的扩展目录加载到浏览器即可。

## 构建

```bash
pnpm build
```
